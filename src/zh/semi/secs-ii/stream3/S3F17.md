## S3,F17  Carrier Action Request

### Direction

M,H->E,reply

### Description

This message requests an action to be performed for a specified carrier. If multi-block, this message must be preceded by the S3,F15/F16 transaction.

### Structure

```text
L,5 
  1. <DATAID> 
  2. <CARRIERACTION> 
  3. <CARRIERID> 
  4. <PTN> 
  5. L,n        n = number of carrier attributes 
       1. L,2 
            1. <CATTRID1> 
            2. <CATTRDATA1> 
       . 
       . 
       n. L,2 
            1. <CATTRIDn> 
            2. <CATTRDATAn> 
```

### Data Item

::: info Exception

If n = 0, then no carrier attributes are included. If CARRIERID is not a zero-length item, then PTN may be omitted (a zero length item). ATTRID and ATTRDATA may be substituted for CATTRID and CATTRDATA respectively.  
ReticlePodLocationID may be used as one of &lt;CATTRID&gt; when the CARRIERACTION is PodRelease and the carrier is not at a Load Port.

:::
