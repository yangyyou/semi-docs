## S3,F18  Carrier Action Acknowledge

### Direction

S,H<-E

### Description

This message acknowledges the carrier action request.

### Structure

```text
L,2 
  1. <CAACK> 
  2. L,n 
       1. L,2 
            1. <ERRCODE1> 
            2. <ERRTEXT1> 
       . 
       . 
       n. L,2 
            1. <ERRCODEn> 
            2. <ERRTEXTn> 
```

### Data Item

::: info Exception

If n = 0, no errors exist.

:::
