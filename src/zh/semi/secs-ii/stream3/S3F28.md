## S3,F28  Change Access Acknowledge

### Direction

S,H<-E

### Description

This message acknowledges the carrier action request.

### Structure

```text
L,2 
  1. <CAACK> 
  2. L,n 
       1. L,3 
            1. <PTN1> 
            2. <ERRCODE1> 
            3. <ERRTEXT1> 
       . 
       . 
       n. L,3 
            1. <PTNn> 
            2. <ERRCODEn> 
            3. <ERRTEXTn> 
```

### Data Item

::: info Exception

If the command is successful, CAACK = 0, and n = 0. If the command was successful for some ports, CAACK  = 6, and n > 0.

:::
