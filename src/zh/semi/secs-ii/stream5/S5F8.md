## S5,F8  List Enabled Alarm Data (LEAD)

### Direction

M,H<-E

### Description

This message is similar to S5,F6 except that it lists only alarms which are enabled.

### Structure

```text
L,m 
  1. L,3 
       1. <ALCD1> 
       2. <ALID1> 
       3. <ALTX1> 
  2. L,3 
  . 
  . 
  m. L,3 
       1. <ALCDm> 
       2. <ALIDm> 
       3. <ALTXm> 
```

::: info Exception

If m = 0, no response can be made. A zero-length item returned for ALCDi or ALTXi means that value does not exist.

:::
