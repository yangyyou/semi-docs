## S5,F3  Enable/Disable Alarm Send (EAS)

### Direction

S,H->E,[reply]

### Description

This message will change the state of the enable bit in the equipment. The enable bit determines if the alarm will be sent to the host. Alarms which are not controllable in this way are unaffected by this message.

### Structure

```text
L,2 
  1. <ALED> 
  2. <ALID>
```

::: info Exception

A zero-length item for ALID means all alarms.

:::
