## S5,F6  List Alarm Data (LAD)

### Direction

M,H<-E

### Description

This message contains the alarm data known to the equipment. There are “m” alarms in the list.

### Structure

```text
L,m 
  1. L,3 
       1. <ALCD1> 
       2. <ALID1> 
       3. <ALTX1> 
  2. L,3 
  . 
  . 
  m. L,3 
       1. <ALCDm> 
       2. <ALIDm> 
       3. <ALTXm> 
```

::: info Exception

If m = 0, no response can be made. A zero-length item returned for ALCDi or ALTXi means that value does not exist.

:::
