## S6,F30   Trace Report Data (TRD)

### Direction

M,H<-E

### Description

Message containing the requested data reports associated with the TRID of trace data report definition.

### Structure

```text
L,3 
  1. <TRID> 
  2. L,n                      (n = number data reports defined for this TRID)  
       1. L,2 
            1. <RPTID1> 
            2. L,m            (m = number of items in this RPTID) 
                 1. <V1> 
                 . 
                 . 
                 m. <Vm> 
       . 
       . 
       n. L,2 
            1. <RPTIDn> 
            2. L,m            (m = number of items in this RPTID) 
                 1. <V1> 
                 . 
                 . 
                 m. <Vm> 
            3. <ERRCODE>
```

::: info Exception

If TRID is unknown, a zero-length list (n = 0) shall be sent. Item 3 (ERRCODE) shall be set to zero length when there is no error.

:::
