## S6,F5  Multi-block Data Send Inquire (MBI)

### Direction

S,H<-E,reply

### Description

If the discrete data report S6F3, F9, F11, F13 can involve more than one block, this transaction must precede the transmission.

### Structure

```text
L,2 
  1. <DATAID> 
  2. <DATALENGTH>
```
