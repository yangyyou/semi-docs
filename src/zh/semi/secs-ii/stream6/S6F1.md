## S6,F1  Trace Data Send (TDS)

### Direction

M,H<-E,[reply]

### Description

This function sends samples to the host according to the trace setup done by S2,F23. Trace is a time-driven form of equipment status.  
Even if S6,F1 is multi-block, it is not preceded by an Inquire/Grant transaction, because the Host S2,F23 is an implicit grant. Some equipment may support only single-block S6,F1, and may refuse an S2,F23 (Trace Initiate Send) message which would cause a multi-block S6,F1.

### Structure

```text
L,4
  1. <TRID> 
  2. <SMPLN> 
  3. <STIME> 
  4. L,n 
       1. <SV1> 
       2. <SV2> 
       . 
       . 
       n. <SVn> 
```

::: info Exception

A zero-length STIME means no value is given and that the time is to be derived from SMPLN along with knowledge of the request.

:::
