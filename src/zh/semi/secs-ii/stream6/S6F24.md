## S6,F24  Request Spooled Data Acknowledgement Send (RSDAS)

### Direction

M,H<-E

### Description

The purpose of this message is to acknowledge the receipt of the Request Spooled Data (S6,F23) and to respond with an appropriate acknowledge code.
