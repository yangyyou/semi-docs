## S2,F39  Multi-block Inquire (DMBI)

### Direction

S,H->E,reply

### Description

If a S2,F23, S2,F33, S2,F35, S2,F45, or S2,F49 message is more than one block, this transaction must precede the message.

### Structure

```text
L,2 
  1. <DATAID> 
  2. <DATALENGTH>
```
