## S2,F14 Equipment Constant Data (ECD)

### Direction

M,H<-E

### Description

Data Response to S2,F13 in the order requested.

### Structure

```text
L,n 
  1. <ECV1> 
  2. <ECV2> 
  . 
  . 
  n. <ECVn>
```

::: info Exception
A zero-length list item for ECVi means that ECIDi does not exist. The list format for this data item is not allowed, except in this case.
:::
