## S2,F37 Enable/Disable Event Report (EDER)

### Direction

S,H->E,reply

### Description

The purpose of this message is for the host to enable or disable reporting for a group of events (CEIDs).

### Structure

```text
L,2 
  1. <CEED>    enable/disable 
  2. L,n    
       1. <CEID1> 
       . 
       . 
       n. <CEIDn> 
```

::: info Exception

A list of zero length following &ltCEED&gt means all CEIDs.

:::
