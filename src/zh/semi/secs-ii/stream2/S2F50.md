## S2,F50  Enhanced Remote Command Acknowledge

### Direction

M,H<-E

### Description

The equipment acknowledges Enhanced Remote Command or reports any error(s). If the command is not accepted due to one or more invalid parameters, (i.e., HCACK = 3), then a list of invalid parameters will be returned containing the parameter name and reason for being invalid.

### Structure

```text
L,2 
  1. <HCACK> 
  2. L,n        # of parameter groups 
       1. L,2 
            1. <CPNAME1> 
            2. <CEPACK1> 
       . 
       . 
       n. L,2 
            1. <CPNAMEn> 
            2. <CEPACKn>
```
