## S2,F42  Host Command Acknowledge (HCA)

### Direction

S,H<-E

### Description

Acknowledge Host command or error. If command is not accepted due to one or more invalid parameters (i.e., HCACK = 3), then a list of invalid parameters will be returned containing the parameter name and reason for being invalid.

### Structure

```text
L,2 
  1. <HCACK> 
  2. L,n                # of parameters
     1. L,2 
          1. <CPNAME1>  parameter 1 name 
          2. <CPACK1>   parameter 1 reason
     . 
     . 
     n. L,2 
          1. <CPNAMEn>  parameter n name 
          2. <CPACKn>   parameter n reason
```

::: info Exception

If there are no invalid parameters, then a list of zero length will be sent for item 2.

:::
