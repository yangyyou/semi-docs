## S2,F16 New Equipment Constant Acknowledge (ECA)

### Direction

S,H<-E

### Description

Acknowledge or error if EAC contains a non-zero error code, the equipment should not change any of the ECIDs specified in S2F15.

### Structure

```text
<EAC>
```
