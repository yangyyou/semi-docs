## S14,F6  GetType Data (GTD)

### Direction

M,H<->E

### Description

### Structure

```text
L,2 
  1. L,n                  n = number of object types 
       1. <OBJTYPE1> 
       . 
       . 
       n. <OBJTYPEn> 
  2. L,2 
       1. <OBJACK> 
       2. L,p               p = number of errors reported  
            1. L,2 
                 1. <ERRCODE1> 
                 2. <ERRTEXT1> 
            . 
            . 
            p. L,2 
                 1. <ERRCODEp> 
                 2. <ERRTEXTp>
```

::: info Exception

If n = 0, there are no owned object types. If p = 0, no errors were detected.

:::
