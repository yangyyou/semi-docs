## S15,F32  Recipe Upload Data

### Direction

M,H<-E

### Description

This message is used to send an execution recipe from a recipe executor.

### Structure

```text
L,4 
  1. <RCPSPEC> 
  2. L,m            (m = # attributes) 
       1. L,2 
            1. <RCPATTRID1> 
            2. <RCPATTRDATA1> 
       . 
       . 
       m. L,2 
            1. <RCPATTRIDm> 
            2. <RCPATTRDATAm> 
  3. <RCPBODY> 
  4. L,2 
       1. <RMACK> 
       2. L,p 
            1. L,2 
                 1. <ERRCODE1> 
                 2. <ERRTEXT1> 
            . 
            . 
            p. L,2 
                 1. <ERRCODEp> 
                 2. <ERRTEXTp>  
```

::: info Exception

p = 0 if and only if RMACK indicates no errors.

:::
