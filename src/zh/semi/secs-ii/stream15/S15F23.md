## S15,F23  Recipe Descriptor Request

### Direction

M,H<->E,reply

### Description

This message is used to request the descriptors of a list of the specified recipes. If multi-block, it must be preceded by the S15,F1/F2 inquire/grant transaction. OBJSPEC is the object specifier of either a recipe namespace of a recipe executor.

### Structure

```text
L,3 
  1. <DATAID> 
  2. <OBJSPEC> 
  3. L,n 
       1. <RCPID1> 
       . 
       . 
       n. <RCPIDn> 
```
