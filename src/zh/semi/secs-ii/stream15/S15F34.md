## S15,F34  Recipe Select Acknowledge

### Direction

M,H<-E

### Description

This message is used to acknowledge the request for recipe selection.

### Structure

```text
L,2 
  1. <RMACK> 
  2. L,p 
       1. L,2 
            1. <ERRCODE1> 
            2. <ERRTEXT1> 
       . 
       . 
       p. L,2 
            1. <ERRCODEp> 
            2. <ERRTEXTp>  
```

::: info Exception

p = 0 if and only if RMACK indicates no errors.

:::
