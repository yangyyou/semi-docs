## S15,F12  Recipe Version Data

### Direction

M,H<->E

### Description

This message contains the recommended version.

### Structure

```text
L,3 
  1. <AGENT> 
  2. <RCPVERS> 
  3. L,2 
       1. <RMACK> 
       2. L,p 
            1. L,2 
                 1. <ERRCODE1> 
                 2. <ERRTEXT1> 
            . 
            . 
            p. L,2 
                 1. <ERRCODEp> 
                 2. <ERRTEXTp> 
```

::: info Exception

If AGENT is a zero-length item in the request, it shall also be a zero-length item in the reply. If it is not zero-length in the request, and it is of zero-length in the reply, then no qualifying recipe was found specific to that equipment. p = 0 if and only if RMACK indicates no errors.

:::
