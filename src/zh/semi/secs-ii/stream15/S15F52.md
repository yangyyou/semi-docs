## S15,F52  Large Recipe Upload Acknowledge (LRUA)

### Direction

S,H<-E

### Description

Acknowledge or error. A returned status of “accepted” means only that the message is understood. The completion of the request is signaled by an event report.

### Structure

```text
<ACKC15>
```

::: info Exception

It is possible to use the ACKC15 code “command will be performed with completion signaled later” for this message.

:::
