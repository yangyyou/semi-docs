## S15,F14  Recipe Create Acknowledge

### Direction

M,H<->E

### Description

This message is used to acknowledge that a recipe has been created or updated with the body sent in the request.

### Structure

```text
L,2 
  1. <RMACK> 
  2. L,2 
       1. L,2 
            1. <ERRCODE1> 
            2. <ERRTEXT1> 
       . 
       . 
       p. L,2 
            1. <ERRCODEp> 
            2. <ERRTEXTp> 
```

::: info Exception

p = 0 if and only if RMACK indicates no errors.

:::
