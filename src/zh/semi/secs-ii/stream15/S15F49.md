## S15,F49  Large Recipe Download Request (LRDR)

### Direction

M,H->E,reply

### Description

This is a request by the host for the equipment to request the download of a recipe via the Stream 13 Data Set Transfer protocol. The Data Set name, DSNAME, is the text string identifier of the recipe, RCPSPEC. The Data Set is subsequently transferred as a Stream with the following internal SECS-II structured data:  

```text
L,4 
  1. <RCPSPEC> 
  2. <DATAID> 
  3. L,m            (m = # of attributes) 
       1. L,2 
            1. <RCPATTRID1> 
            2. <RCPATTRDATA1> 
       . 
       . 
       m. L,2 
            1. <RCPATTRIDm> 
            2. <RCPATTRDATAm> 
  4. <RCPBODY> 
```

### Structure

```text
L,2 
  1. <DSNAME> 
  2. <RCPOWCODE> 
```
