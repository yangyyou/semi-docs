## S15,F33  Recipe Select Request

### Direction

M,H->E,reply

### Description

This message is used to request the selection of one or more execution recipes. If multi-block, it shall be preceded by the S15,F1/S15,F2 inquire/grant transaction.

### Structure

```text
L,3 
  1. <DATAID> 
  2. <RESPEC> 
  3. L,r                        (r = # selections) 
       1. L,2 
            1. <RCPID1>         (1st recipe selection) 
            2. L,p              (p = # parameter settings for 1st recipe) 
                 1. L,2 
                      1. <RCPPARNM11> 
                      2. <RCPPARVAL11> 
                 . 
                 . 
                 p. L,2 
                      1. <RCPPARNM1p> 
                      2. <RCPPARVAL1p> 
       . 
       . 
       r. L,2 
            1. <RCPIDr>         (rth recipe selection) 
            2. L,s              (s = # parameter settings for rth recipe) 
                 1. L,2 
                      1. <RCPPARNMr1> 
                      2. <RCPPARVALr1> 
                 . 
                 . 
                 s. L,2 
                      1. <RCPPARNMrs> 
                      2. <RCPPARVALrs>
```

::: info Exception

If the list of parameter settings for a recipe selection is of zero length, then no parameter settings are specified for the corresponding recipe.

:::
