## S15,F4  Recipe Namespace Action Acknowledge

### Direction

M,H<->E

### Description

This message is used to confirm whether the requested action was completed successfully or to provide error information otherwise.

### Structure

```text
L,2 
  1. <RMACK> 
  2. L,p 
       1. L,2 
            1. <ERRCODE1> 
            2. <ERRTEXT1> 
       . 
       . 
       p. L,2 
            1. <ERRCODEp> 
            2. <ERRTEXTp>
```

::: info Exception

p = 0 if and only if RMACK indicates no errors.

:::
