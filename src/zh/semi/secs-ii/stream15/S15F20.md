## S15,F20  Recipe Rename Acknowledge

### Direction

M,H<->E

### Description

This message acknowledges the request to copy or rename a recipe and indicates whether the action was successfully performed or errors that occurred.

### Structure

```text
L,2 
  1. <RMACK> 
  2. L,p 
       1. L,2 
            1. <ERRCODE1> 
            2. <ERRTEXT1> 
       . 
       . 
       p. L,2 
            1. <ERRCODEp> 
            2. <ERRTEXTp>
```

::: info Exception

p = 0 if and only if RMACK indicates no errors.

:::
