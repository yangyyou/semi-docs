## S15,F25  Recipe Parameter Update Request

### Direction

M,H<->E,reply

### Description

This message is used to update the variable parameter definitions for a specific agent. If multi-block, it must be preceded by the S15,F1/F2 inquire/grant transaction.

### Structure

```text
L,4 
  1. <DATAID> 
  2. <RMNSSPEC> 
  3. <AGENT> 
  4. L,n 
       1. L,3 
            1. <RCPPARNM1> 
            2. <RCPPARVAL1> 
            3. <RCPPARRULE1> 
       . 
       . 
       n. L,3 
            1. <RCPPARNMn> 
            2. <RCPPARVALn> 
            3. <RCPPARRULEn>  
```
