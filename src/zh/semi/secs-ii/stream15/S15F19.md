## S15,F19  Recipe Retrieve Request

### Direction

S,H<->E,reply

### Description

This message is used to request that a recipe be copied to, or renamed to, a recipe with a new identifier.

### Structure

```text
L,3 
  1. <RCPSPEC> 
  2. <RCPRENAME>
  3. <RCPNEWID>
```
