## S15,F13  Recipe Create Request

### Direction

M,H<->E,reply

### Description

This message is used to create or modify a recipe body. If multi-block, it shall be preceded by the S15,F1/F2 inquire/grant transaction.

### Structure

```text
L,5 
  1. <DATAID> 
  2. <RCPUPDT> 
  3. <RCPSPEC> 
  4. L,m 
       1. L,2 
            1. <RCPATTRID1> 
            2. <RCPATTRDATA1> 
       . 
       . 
       m. L,2 
            1. <RCPATTRIDm> 
            2. <RCPATTRDATAm> 
  5. <RCPBODY>
```

::: info Exception

RCPBODY may be of zero length.

:::
