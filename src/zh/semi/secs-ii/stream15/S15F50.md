## S15,F50  Large Recipe Download Acknowledge (LRDA)

### Direction

S,H<-E

### Description

Acknowledge or error. A returned status of “accepted” means only that the message is understood. Upon completion of the large recipe download request (Stream 13 Data Set transfer scenario) the equipment initiates a separate verification transaction (S15,F53/S15,F54) that provides the result of the verification.
