## Stream 15 Recipe Management

The functions in this stream are used requesting information and operations concerning recipes, recipe namespaces, and recipe executors. A recipe is an object that is transferred in sections, where a section consists of either recipe attributes, agent-specific dataset attributes, or the body of the recipe. An attribute is information concerning the recipe body, the recipe as a whole, or the application of the recipe. An attribute consists of an attribute name/attribute value pair.
