## S7,F26  Formatted Process Program Data (FPD)

### Direction

M,H<->E

### Description

This message transfers a process program in response to a request for the PPID. The values of MDLN and SOFTREV are obtained from the PCD used to generate the process program.

### Structure

```text
L,4 
  1. <PPID> 
  2. <MDLN> 
  3. <SOFTREV> 
  4. L,c              (c = Number of Process Commands) 
       1. L,2  
            1. <CCODE> 
            2. L,p    (p = Number of Parameters) 
                 1. <PPARM1> 
                 . 
                 . 
                 p. <PPARMp> 
       2. L,2 
       . 
       . 
       c. L,2
```

::: info Exception

A zero length list indicates the request was denied.

:::
