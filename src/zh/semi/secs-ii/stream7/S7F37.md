## S7,F37  Large Process Program Send (LPPS)

### Direction

M,H<->E,reply

### Description

This is a request to send a process program via the Data Set Transfer protocol. The Data Set name, DSNAME, is the text string identifier of the process program, PPID. The Data Set is subsequently transferred as a Stream with the following internal SECS-II structured data:  
&lt;PPBODY>&gt;

### Structure

```text
<DSNAME> 
```
