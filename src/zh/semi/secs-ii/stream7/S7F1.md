## S7,F1  Process Program Load Inquire (PPI)

### Direction

S,H<->E,reply

### Description

This message is used to initiate the transfer of a process program or to select from stored programs. The message may be used to initiate the transfer of an unformatted process program (S7,F3/S7,F4) or a formatted process program (S7,F23/S7,F24), (S7,F31/S7,F32).

### Structure

```text
L,2 
  1. <PPID> 
  2. <LENGTH>
```
