## S7,F6  Process Program Acknowledge (PPA)

### Direction

M,H<->E

### Description

This message is used to transfer a process program.

### Structure

```text
L,2 
  1. <PPID> 
  2. <PPBODY>
```

::: info Exception

A zero-length list means request denied.

:::
