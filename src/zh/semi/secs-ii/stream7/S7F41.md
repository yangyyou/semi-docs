## S7,F41  Large Process Program Request(LPPR)

### Direction

S,H<->E,reply

### Description

This message is used to request the transfer of a process program via the Stream 13 Data set Transfer protocol. The Data Set name, DSNAME, is the text string identifier of the process program, PPID. The Data Set is subsequently transferred as a Stream with the following internal SECS-II structured data:  
&lt;PPBODY&gt;

### Structure

```text
<DSNAME>
```
