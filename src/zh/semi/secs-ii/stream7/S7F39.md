## S7,F39  Large Formatted Process Program Send (LFPPS)

### Direction

S,H<->E,reply

### Description

This is a request to send a formatted process program via the Data Set Transfer Protocol. The Data Set name, DSNAME, is the text string identifier of the process program, PPID. The Data Set is subsequently transferred as a Stream with the following internal SECS-II structured data:

```text
L,4 
  1. <PPID> 
  2. <MDLN> 
  3. <SOFTREV> 
  4. L,c    
       1. L,2 
            1. <CCODE> 
            2. L,p   
(c = Number of Process Commands) 
(p = Number of Parameters) 
                 1. <PPARM1> 
                 . 
                 . 
                 p. <PPARMp> 
       2. L,2 
       . 
       . 
       c. L,2 
```

### Structure

```text
<DSNAME>
```
