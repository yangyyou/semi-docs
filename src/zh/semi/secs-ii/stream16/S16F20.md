## S16,F20  PRGetAllJobs Send

### Direction

S,H<-E

### Description

Returns the requested list of process jobs.

### Structure

```text
L,m                 [m = # jobs in the list]
  1. L,2 
       1. <PRJOBID1> 
       2. <PRSTATE1> 
  . 
  . 
  m. L,2 
       1. <PRJOBIDm> 
       2. <PRSTATEm> 
```

::: info Exception

If m = 0, then no process jobs are running or waiting to run.

:::
