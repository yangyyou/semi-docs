## S16,F12  PRJobCreateEnh Acknowledge

### Direction

S,H<-E

### Description

This message acknowledges the request and reports any errors in the creation of a process job.

### Structure

```text
L,2 
  1. <PRJOBID> 
  2. L,2 
       1. <ACKA> 
       2. L,n 
            1. L,2 
                 1. <ERRCODE1> 
                 2. <ERRTEXT1> 
            . 
            . 
            n. L,2 
                 1. <ERRCODEn> 
                 2. <ERRTEXTn> 
```

::: info Exception

If n = 0, no errors exist.

:::
