## S16,F25  PRJobSetStartMethod

### Direction

S,H->E,reply

### Description

Used to request to change the start method (Manual start or Automatic start) for one or more process jobs.

### Structure

```text
L,2 
  1. L,m                 [m = # of jobs; m >= 1] 
       1. <PRJOBID1> 
       . 
       . 
       m. <PRJOBIDm> 
  2. <PRPROCESSSTART> 
```
