## S16,F28  Control Job Command Acknowledge

### Direction

S,H<-E

### Description

Indicates success or failure of command request to a control job. If applicable ERRTEXT shall contain information on specific command parameter names or values that caused the error.

### Structure

```text
L,2 
  1. <ACKA> 
  2. L,2 
       1. <ERRCODE> 
       2. <ERRTEXT> 
```

::: info Exception

2 . L,2 IS L,0 if no errors.

:::
