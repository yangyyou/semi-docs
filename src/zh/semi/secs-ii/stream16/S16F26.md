## S16,F26  PRJobSetStartMethod Acknowledge

### Direction

S,H<-E

### Description

Acknowledges request to set job start method and indicates any errors. ERRTEXT will contain the identifiers of any process jobs that did not accept the new process start method.

### Structure

```text
L,2 
1. L,k                             [k = # of jobs which are referenced in the request  
       1. <PRJOBID1>                and that are now in the specified start method] 
       . 
       . 
       k <PRJOBIDk> 
  L,2 
    1. <ACKA> 
    2. L,n                         [n = # pf jobs for which the request failed] 
         1. L,2 
              1. <ERRCODE1> 
              2. <ERRTEXT1> 
         . 
         . 
         n. L,2 
              1. <ERRCODEn> 
              2. <ERRTEXTn> 
```
