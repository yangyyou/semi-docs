## S16,F24  PRJobSetRecipeVariable Acknowledge

### Direction

S,H<-E

### Description

Indicate the status of the request to set recipe variables. ERRTEXT will contain the RCPPARNM value for parameters that could not be reset.

### Structure

```text
L,2 
  1. <ACKA> 
  2. L,n 
       1. L,2 
            1. <ERRCODE1> 
            2. <ERRTEXT1> 
       . 
       . 
       n. L,2 
            1. <ERRCODEn> 
            2. <ERRTEXTn>
```

::: info Exception

If n = 0, no errors exist.

:::
