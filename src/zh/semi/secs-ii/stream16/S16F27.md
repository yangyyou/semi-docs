## S16,F27  Control Job Command Request

### Direction

S,H->E,reply

### Description

Send a control job command to a control job.

### Structure

```text
L,3 
  1. <CTLJOBID> 
  2. <CTLJOBCMD> 
  3. L,2 
       1. <CPNAME> 
       2. <CPVAL>
```

::: info Exception

3 . L,2 IS L,0 for commands that do not need parameters.

:::
