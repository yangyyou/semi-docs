## S16,F5  Process Job Command Request (PRJCMDR)

### Direction

M,H->E,reply

### Description

Send a job control command to a processing job.

### Structure

```text
L,4 
  1. <DATAID> 
  2. <PRJOBID> 
  3. <PRCMDNAME> 
  4. L,n 
       1. L,2 
            1. <CPNAME1> 
            2. <CPVAL1> 
       . 
       . 
       n. L,2 
            1. <CPNAMEn> 
            2. <CPVALn> 
```

::: info Exception

The CPNAME, CPVAL pairs are command parameter identifiers and values; n = 0 is valid for some commands (PRCMDNAME).

:::
