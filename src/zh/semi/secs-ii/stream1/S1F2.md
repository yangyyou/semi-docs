## S1,F2  On Line Data (D)

### Direction

S,H<->E

### Description

Data signifying that the equipment is alive.

### Structure

```text
L,2 
  1. <MDLN> 
  2. <SOFTREV>
```

::: info Exception
The host sends a zero-length list to the equipment.
:::
