## S1,F11  Status Variable Namelist Request (SVNR)

### Direction

S,H->E,reply

### Description

A request to the equipment to identify certain status variables.

### Structure

```text
L,n 
  1. <SVID1> 
  . 
  . 
  n. <SVIDn>
```

::: info Exception
A zero length means report all SVIDs.
:::
