## S1,F14 Establish Communications Request Acknowledge (CRA)

### Direction

S,H<->E,reply

### Description

Accept or deny Establish Communications Request (S1,F13). MDLN and SOFTREV are on-line data and are valid only if COMMACK = 0.

### Structure

```text
L,2 
  1. <COMMACK> 
  2. L,2 
       1. <MDLN> 
       2. <SOFTREV> 
```

::: info Exception
The host sends a zero-length list for item 2 to the equipment.
:::
