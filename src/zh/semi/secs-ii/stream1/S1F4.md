## S1,F4  Selected Equipment Status Data (SSD)

### Direction

M,H<-E

### Description

The equipment reports the value of each SVID requested in the order requested. The host remembers the names of values requested.

### Structure

```text
L,n 
  1. <SV1> 
  . 
  . 
  n. <SVn>
```

::: info Exception
A zero-length list item for SVi means that SVIDi does not exist.
:::
