## S12,F5  Map Transmit Inquire (MAPTI)

### Direction

S,H<-E,reply

### Description

Used to prepare the host for map transmission. S12,F5 must precede all S12,F7–8, F9–10, & F11–12 transactions.

  2. <IDTYP>
