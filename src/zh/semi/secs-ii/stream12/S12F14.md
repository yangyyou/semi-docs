## S12,F14  Map Data Type 1 (MD1)

### Direction

M,H->E

### Description

Used to send map data from the host to the equipment in row or column format.

### Structure

```text
L,3
  1. <MID> 
  2. <IDTYP> 
  3. L,n 
       1. L,2 
            1. <RSINFx1 RSINFy1RSINd> 
            2. <BINLT...> 
       2. L,2 
       . 
       . 
       n. L,2 
            1. <RSINFn> 
            2. <BINLTn> 
```
