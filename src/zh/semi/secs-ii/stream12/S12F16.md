## S12,F16  Map Data Type 2 (MD2)

### Direction

M,H->E

### Description

Used to send map data from the host to the equipment in array format.

### Structure

```text
L,4
  1. <MID> 
  2. <IDTYP> 
  3. <STRPx STRPy> 
  4. <BINLT>
```

::: info Exception

A zero-length list returned means no such MID.

:::
