## S12,F9  Map Data Send Type 2 (MDS2)

### Direction

M,H<-E,reply

### Description

Used to send map data from the equipment in array format. If S12,F9 is multi-block, it must be preceded by the S12,F5/S12,F6 Inquire/Grant transaction.

### Structure

```text
L,4 
  1. <MID> 
  2. <IDTYP> 
  3. <STRPx STRPy> 
  4. <BINLT...> 
```
